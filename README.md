# M&J Core v0.5.1 — Responsive 4x2 · Mensajes de Mia

Versión adaptada para un monitor de **4 bloques de ancho por 2 de alto**.

## Cambios de la v0.5.1
- Actualizados los mensajes personalizados del detector de Mia.
- Protegidos los mensajes, la zona del detector y las listas de tareas para que el actualizador no los sobrescriba.
- Número de versión sincronizado en `README.md`, `version.json`, `manifest.json` y `mjcore/core/config.lua`.

## Diseño 4x2
- Escritorio en cuadrícula 2x2.
- Dos páginas para las 8 aplicaciones.
- Botones grandes y táctiles.
- Escala 0.5 para aprovechar la pantalla.
- Inventario y tareas adaptados a la altura disponible.
- Mantiene el detector de Mia y sus coordenadas configuradas.

## Instalación
Copia todo el contenido en la raíz del Advanced Computer y reinicia.
